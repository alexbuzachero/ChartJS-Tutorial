The birth of React

Back in 2011, Facebook started to face several issues with code maintenance. Also, because of the Facebook Ads app got an increasing number of several features, was necessary more people to keep it running. Not only this, their app became very hard to handle, as they faced a lot of cascading updates.

To solve it Jordan Walke built a prototype that made the process more efficient, and this marks the birth of React.js.

What is React and how works?

It is very simple to understand how efficient is React, React is a declarative and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

To simplify how use React in your web page, look the image below:

In this picture representation each box color represents one React component, and React give you the freedom to work in each component in parallel, and every change that you make inside each component will update the webpage automatically and online.

Inside the React component

Inside each component, there is a bit of code that represents a piece of the page, and the way that you call each component is very similar the way that we call a function using Javascript.

User Stories

To have a best understand need to develop this prototype was necessary to create the user stories. Also, what is a user story??

A user story is a tool used in Agile software development to capture a description of a software feature from an end-user perspective. The user story describes the type of user, what they want and why. A user story helps to create a simplified description of a requirement.

Rental Moose

Rental Moose is a Website that provides an opportunity to Landlord offer house to rent, and on the other side, the future tenant can access our website to consult a plenty of houses available to rent. However, the main focus in our Term 3 project is Data Visualization, because of that the tenant need to fill a form and through this data, we will create the tenant score. Also, that information will be essential to create the dashboard that the Landlord can analyze the score of each possible tenant. In this case, our data visualization will be detailing about different criteria that were used to reach that Tenant score and representing in graph format.

Project User Stories

As a New user, I want to Register a new account and I can use the services for either finds a place to rent or rent out my existing place.

As a Landlord, I want to see who is interesting to rent my place and I can check the tenant Risk score and using that decide who has the best tenant profile.

As a Tenant, I want to see a list of properties that are available for rent and I can apply to rent the place that I want.
